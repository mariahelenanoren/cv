The Non-anonymous Autonomous Society

A digital exhibition about the possible threat of living in a society where AI is implemented for survelliance purposes

Artificial intelligens is a reatively new technology which can be with many different purposes. One of the being survelliance. On this subject I created six different pieces which in different ways discusses the implementation of AI in survelliance. These were then presented as a digital exhibition on a website I created. This project was my graduation project at HDK, at which time I did not know very much about coding and wrote it in a completely different way than I would have today, which is why the code is not public.
