Väder – Swedish weather forecast

A website which displays weather forecasts for Swedish cities and towns

The website uses SMHI's (the Swedish Meteorological and Hydrological Institute) Open Data API to present accurate weather forecasts. Locations can be favorited by clicking the heart on the forecast page. All favorites are presented on the index page with the current degrees and weather condition. As the seasons change so does the websites design, with the background color and image being switched to fit the time of year.
